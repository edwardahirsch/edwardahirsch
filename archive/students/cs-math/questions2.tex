\centerline{\large\bfseries Вопросы к зачету по курсу <<Информатика>>}
\centerline{\small 110--113 гр., весенний семестр 2003 г.}

\begin{enumerate}
\item Праволинейные грамматики; теорема о приведении к нормальной форме.
\item Теорема об эквивалентности детерминированных и недетерминированных
      конечных автоматов.
\item Конечные автоматы, регулярные выражения и праволинейные грамматики:
      теорема об эквивалентности.
\item Лемма о разрастании для регулярных языков; языки, не являющиеся
      регулярными.
\item Замкнутость класса регулярных языков относительно теоретико-множественных
      операций. Разрешимые проблемы, связанные с регулярными языками.
\item Бесконтекстные грамматики; теорема о приведении к нормальной форме.
\item Автоматы с магазинной памятью; построение автомата,
      принимающего данный бесконтекстый язык.
\item Лемма о разрастании для бесконтекстных языков.
      Разрешимые алгоритмические проблемы, связанные с бесконтекстными языками.
\item Контекстно-зависимые языки.
\item Рекурсивно-перечислимые языки.
      Существование рекурсивно-перечислимого, но не рекурсивного языка.
\item Классы P и NP. Сводимости и полнота.
\item Алгоритмы, использующие случайные числа.
      Вероятностный алгоритм, распознающий составные числа.
\item Приближенный алгоритм для задачи о рюкзаке.
\item Приближенные алгоритмы для задачи о коммивояжере.
\item Приближенный алгоритм для задачи о покрытии множествами.
\item Приближенный алгоритм для задачи о кратчайшей общей надпоследовательности.
\item Алгоритм для задачи о поиске подстроки. 
\item Дискретное преобразование Фурье (ДПФ).
      Теорема об отрицательно обернутой свертке.
      Вычисление ДПФ.
\item Алгоритм Ш\"енхаге-Штрассенa
      (без вычисления ДПФ, но с ``простым'' рекурсивным алгоритмом).
\item Параллельный алгоритм для задачи о достижимости в графе.
\item Параллельный алгоритм, находящий
      максимальное по включению независимое множество.
\end{enumerate}
