\noindent{}Программа экзамена по с/к ``Эффективные алгоритмы-I'' (весна 2000 г.):
\begin{enumerate}
\item Алгоритм Штрассена для умножения матриц.
      Умножение булевых матриц. [АХУ] 
\item Алгоритм Шенхаге-Штрассена для умножения чисел. [АХУ]
\item Метод отпечатков пальцев: сравнение строк на расстоянии,
      поиск подстроки, проверка умножения матриц. [MR]
\item Приближенные алгоритмы, основанные на полуопределенном
      программировании. [GW]
\item Приближенный алгоритм подсчета мощности объединения множеств. [MR]
\item Алгоритм кэширования. [MR]  
\item Алгоритм k официантов. [KP] 
\item Алгоритм определения расстояний между всеми парами вершин графа. [MR]
\end{enumerate}

    М.: Мир, 1979.

    Cambridge University Press, 1995.

    Algorithms for Maximum Cut and Satisfiability Problems Using

\item[{[KP]}] E.Koutsoupias and C. Papadimitriou. On the k-server conjecture.
    Journal of the ACM, 42(5):971--983, 1995.\\http://www.cs.ucla.edu/$\sim$elias/publications/paper-kp95.ps
\end{enumerate}
