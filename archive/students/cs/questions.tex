\bigskip
\centerline{\bfseries Часть I. Основные алгоритмы и структуры данных}

\begin{enumerate}
\item Массовая задача. Тезис Черча. Машина Тьюринга.
\item Равнодоступная адресная машина. 
      Время работы алгоритма и используемая память.

\item Рекурсия, ее реализация на компьютере и способы избавления от нее
      (на примере задачи о рюкзаке).

\item Структуры данных (массив, файл, очередь, стек, списки, словарь).
\item АВЛ-деревья. Реализация операций над ними.
\item B-деревья. Реализация операций над ними. B${}^+$-деревья.

\item Сложность рекурсивных алгоритмов.
\item Алгоритм Штрассена для умножения матриц над кольцом. 
      Умножение булевых матриц.
\item Сортировка на четырех лентах.
\item Нахождение пары ближайших точек на плоскости. 

\item Способы представления графа в машине. Поиск в глубину и ширину.
\item Топологическая сортировка.
\item Разбиение графа на компоненты сильной связности.
\item Построение минимального остовного дерева.
\item Нахождение кратчайших путей: алгоритмы Дейкстры и Беллмана-Форда.
\item Полилогарифмический по памяти алгоритм для определения 
      длины кратчайшего пути.
\item Нахождение кратчайших путей между всеми парами вершин:
      алгоритм Флойда-Уоршолла.
\item Задача о максимальном потоке. 
      Лемма о максимальном потоке и минимальном сечении.
      Общая схема алгоритма Форда-Фалкерсона.
\item Варианты реализации алгоритма Форда-Фалкерсона: случай
      целочисленных пропускных способностей; алгоритм Эдмондса-Карпа.
\item Метод проталкивания предпотока.
\item Применение алгоритма Форда-Фалкерсона для нахождения 
      максимального паросочетания в двудольном графе.

\item Лексикографическая сортировка.
\item Изоморфизм деревьев (без алгоритма лексикографической сортировки).

\item HeapSort.
\item QuickSort. Randomized QuickSort.
\item Поиск $k$-го элемента в массиве.

\item Дискретное преобразование Фурье (ДПФ).
      Теорема об отрицательно обернутой свертке.
      Вычисление ДПФ.
\item Алгоритм Ш\"енхаге-Штрассенa
      (без вычисления ДПФ, но с ``простым'' рекурсивным алгоритмом).

\end{enumerate}

\bigskip
\centerline{\bfseries Часть II. Современные алгоритмы и элементы теории сложности}

\begin{enumerate}\setcounter{enumi}{28}
\item Рекурсивно-перечислимые и рекурсивные множества.
      Многоленточные машины Тьюринга. Универсальная машина Тьюринга.
      Существование рекурсивно-перечислимого, но не рекурсивного множества.

\item Классы $\mathcal{P}$, $\mathcal{NP}$, $\mathcal{PSPACE}$, $\mathcal{EXP}$.
      Полиномиальные свед\'ения. $\mathcal{NP}$-полные задачи.
      Задача об ограниченной остановке и другие примеры
      $\mathcal{NP}$-полных задач.

\item Классы DTime$(f(n))$, доказательство того, что они различны.

\item Вероятностные алгоритмы. Алгоритм для выяснения того,
      что число является составным.

\item Общая схема кодирования с открытым ключом.
      Односторонние функции, сильно односторонние функции,
      функции ``с секретом''. RSA. Электронные подписи.

\item Приближенные алгоритмы. Приближенные алгоритмы для задач
      о рюкзаке, покрытии множествами, наименьшей общей
      надпоследовательности, наименьшего вершинного покрытия,
      задачи о коммивояжере в метрическом пространстве.

\item Рисование планарного графа.

\item Нахождение пары пересекающихся отрезков на плоскости.

\item Построение выпуклой оболочки на плоскости.

\item Конечные автоматы и регулярные множества.
      Определение принадлежности регулярному множеству.
      Поиск подстроки.

\item Параллельные алгоритмы. Модели параллельных вычислений.
      Принцип Брента. Параллельные алгоритмы для умножения булевых матриц,
      достижимости в графе, вычисления определителя, арифметических
      операций над целыми числами.

\item Алгоритмы, обрабатывающие вход по мере поступления.
      Задача о кэшировании.

\end{enumerate}
